Downloading Dillo

Dillo is following an evolving software model where every new version shall be better than the previous one; there's no place for unstable releases, so just keep up with the latest one: dillo-3.0.5

Steps:

  1. Get the latest tarball (marked with a green arrow) from the source archive.
  2. Follow build directions from the README file.
